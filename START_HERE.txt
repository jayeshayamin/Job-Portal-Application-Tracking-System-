🔧 SETUP FOR YOUR PLATFORM:

WINDOWS (XAMPP):
1. Place in: C:\xampp\htdocs\Job-Portal-Application-Tracking-System-\
2. Start Apache & MySQL in XAMPP Control Panel
3. Import schema.sql in phpMyAdmin
4. Run: php seed_mysql.php
5. Open: http://localhost/Job-Portal-Application-Tracking-System-/

WINDOWS (WAMP):
1. Place in: C:\wamp\www\Job-Portal-Application-Tracking-System-\
2. Start MySQL & Apache via WAMP
3. Import schema.sql in phpMyAdmin
4. Run: php seed_mysql.php
5. Open: http://localhost/Job-Portal-Application-Tracking-System-/

LINUX:
1. Extract to: /var/www/html/Job-Portal-Application-Tracking-System-/
2. Run: mysql -u root -p < schema.sql
3. Run: php seed_mysql.php
4. Open: http://localhost/Job-Portal-Application-Tracking-System-/

MAC (MAMP):
1. Extract to: /Applications/MAMP/htdocs/Job-Portal-Application-Tracking-System-/
2. Start MAMP servers
3. Import schema.sql in phpMyAdmin
4. Run: php seed_mysql.php
5. Open: http://localhost:8888/Job-Portal-Application-Tracking-System-/

See INSTALLATION.md for detailed platform-specific instructions!
